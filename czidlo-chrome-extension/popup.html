CZIDLO urn:nbn:cz

Toto rozšíření detektuje identifikátory urn:nbn:cz na stránce a po najetí myší nad nimi zobrazuje dodatečné informace včetně odkazů do digitálních knihoven. Informace získává z API Resolveru URN:NBN (CZIDLO) provozovaného Národní knihovnou České Republiky.
Více informací naleznete na GitHub stránce projektu.
